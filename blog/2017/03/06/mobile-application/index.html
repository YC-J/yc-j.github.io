アプリ制作の概要と流れ

Mar 6th, 2017 9:38 am

流れ

  • 企画 → 開発 → テスト → 公開 → 保守

OS

  • iOS
  • Android
  • Windows Mobile

iOS

  • 開発環境
    • Xcode

  • 言語
    • Objective-C
    • Swift
    • 2014年6月2日初公開以後、最初はObjective-Cが圧倒的に多かったがだんだんSwiftがObjective-Cのシェアを奪っている
    • 言語ランキング
      • http://www.tiobe.com/tiobe-index/
  • 条件
    • Apple Storeでの公開には年間11,800円
  • Developer Site
    • Apple Developer
    • https://developer.apple.com/

Android

  • 開発環境
    • Android Studio

    • Eclipse

  • 言語
    • Java
  • 条件
    • Google Playでの公開には初回のみ25ドル
  • Developer Site
    • Google Developer
    • https://developers.google.com/

アプリの種類

Native App

Hybrid App

  • 開発ツール
    • Apache Cordova
    • Adobe PhoneGap
    • Ionic Framework
    • Monaca
  • https://jq-jo.github.io/blog/2016/08/31/Hybrid-Native-Responsive/

mBaaS (Moblie Backend as a Service)

  • mBaaSはApplicationに要求されるいろんなBackendの機能をまとめて提供してくれるサービス
  • 最もよく使われるサービス
    • AWS Mobile
    • Firebase
  • 提供サービス
    • ユーザ認証
    • データベース
    • ストレージ
    • プッシュ通知
    • analytics
    • etc…

サンプルコード

  • Webview
  • Http通信
  • Tab bar Controller

  • iOS
    • iTunes Connect(https://itunesconnect.apple.com)
  • Android
    • Google Play Developer Concole(https://play.google.com/apps/publish)
  • https://jq-jo.github.io/blog/2016/10/26/mobile-app-sinsa/
